01. Background & Concept

開発背景とコア・コンセプト

現代のデジタルアートの課題

  • 「静観」から抜け出せない体験: 多くのデジタル表現が画面の向こう側の鑑賞にとどまり、身体や感情の直感的なアウトプットと切り離されている。
  • 双方向性の希薄さ: 単にマウスをクリックする、決められたボタンを押すだけといった、単調で機械的なインタラクションの限界。
  • 感情の未反映: ユーザーのリアルタイムな「心のゆらぎ」がデジタル表現の挙動に結びついていない。

KinetiChordの解答

身体全体を「楽器」に、表情を「光」に変え、空間に自らの存在を共鳴させる。

  • 極限のマルチモーダル融合: ウェブカメラによる「表情分析 ＆ 身体モーション追跡」と「マイク音圧解析」を極限までブレンド。
  • 「音と光をまとう」新体験: 自身の身体の速度そのものが粒子エフェクトやシンセの和音をダイナミックに変化させる超感覚フィードバック。

03. Core Feature I

身体を完璧にトレースする「キネティック・スケルトン」

二律駆動 (Dual-Mode) アバター

  • リアルタイム・トラッキング: 表情認識により得られた「顔の座標」と「手のモーション重心」をミリ秒単位で追従（Lerpによる滑らかな補間）。
  • 自律ダンス・デフォルト回帰: カメラからロスト、または非検出のときは、海藻や水中ダンサーのように優雅にたゆたう「自律ダンスポーズ」へ滑らかに移行。不自然な「直立不動」やフリーズを根絶。

3Dシリンダー投影・幾何数学

  • 立体的な脚部ボリューム復元: 股関節から足首に至るまでの骨格を「3Dテーパーシリンダー（肉厚な円柱）」として数理モデリング。
  • シネマティック回転カメラ: 自転するアングル（左右）とたゆたうアングル（上下）をリアルタイムに計算し、3D空間上のアバター座標を2Dスクリーン座標へ完璧に射影。3D表現の真髄を引き出します。

04. Core Feature II

動きに連動するダイナミック・パーティクル

① 手速度追跡 ＆ 物理軌跡バースト

左右の手の位置のフレーム間差分を監視し、移動速度（`speed > 8.0`）を算出：

  • 軌跡バースト粒子: 素早く手を振ると、手の進行ベクトルを継承した3〜8個の輝く軌跡粒子が鋭く射出。
  • 直感的な爽快感: 画面上で実際に光の球を手から「投げ飛ばす」ようなリアルな物理インタラクション。

② 全身モーションスパーク ＆ 星座ライン

  • 虹色スパーク (Sparks): ウェブカメラ全体の動きの激しさ（`activeMotionPoints.length`）が閾値（120）を超えると、骨格パーツから極めて寿命が短く瞬く「レインボーの火花」がポポポと吹き出します。
  • 動的星座ライン脈動: 全身の動きに連動し、星座の接続線（areAdjacentBodyParts）が太く明るく脈動。さらに、動きが激しいときは線の接続限界距離が自動拡張されます。

05. Core Feature III

音を極限まで変化させる「Extreme Sound FX」

対象の感情 ビジュアルエフェクト 極限音響モジュレーション (Extreme Sound FX)
😊 喜び (Happy) 黄金（ゴールド）の気泡オーラが優雅に上昇。 アルペジエーター・スイープ: ローパスを9,500Hzに完全解放。120ms周期でコード音がオクターブ上にステップし流れるようなきらめき音色。
😢 哀しみ (Sad) 悲しみの涙粒子が下へポトポト滴り落ちる。 深海フェードアウト: 音量最大85%カット。ローパスを150Hzの深海極限状態にし、ディレイ帰還率88%のエコーで哀愁がどこまでも溶けていく。
😡 怒り (Angry) 真紅（レッド）の火花が激しい初速で四散。 爆音ディストーション: クリーン音を完全0%に遮断。WaveShaperによる歪み量を極限（350）まで増幅、マイク音圧連動的爆音ゲインと悲鳴ピッチジッター。
😮 驚き (Surprised) シアンの衝撃パルス粒子が全方位に大爆発。 15Hz高速ステレオ往復パン: 音像が1秒間に15回左右に揺さぶられ、金属音が鳴り響く極小の0.05秒スラップバックエコーが襲うサイケデリック音響。
😐 平穏 (Neutral) エメラルドグリーンの光の霧がたゆたう。 心地よい5声ドローン音: 5つのシンセ（サイン波＆三角波）が、脳を心地よくリラックスさせるドローン持続音を優しく響かせます。

06. Tech Stack & Optimization

徹底したVanilla実装と超高速パフォーマンス最適化

技術スタックの一覧

HTML5 / Canvas 2D Vanilla CSS Vanilla JS (ES6+) face-api.js (TensorFlow) Web Audio API Docker GCP Cloud Run

重厚なUIフレームワーク（React等）を排し、ブラウザ機能に極限まで密着した「Pure Vanilla」で構築。圧倒的なロード速度と描画レスポンスを実現。

60FPSを維持するためのチューニング

  • モーション検出の超軽量化: 解析解像度を `48x36` ピクセルに縮小。パフォーマンス消費を事実上ゼロにクランプしつつ、手の重心を見事に捉えます。
  • 非同期モデルロード: TensorFlowの表情モデル（約5MB）はアプリ起動と同時に非同期読み込み。UIの操作を1秒も阻害しません。
  • 厳格なパーティクル管理: 最大描画数を `650` に制御。寿命を迎えた粒子を即時ガベージコレクション。

07. Deployment & Verification

Docker ＆ Google Cloud Run への自動デプロイ構成

デプロイの仕組み

静的Webサーバー（Nginx）を含む超軽量な `Alpine-based Dockerfile` を定義し、GCP Cloud Runへ直結。

  • 秒速のスケールアウト: アクセス急増時もサーバーレスで自動スケーリング。常時起動（ミニマムインスタンス）設定により、コールドスタート時の遅延を根絶しています。
  • HTTPS常時強制: カメラ・マイク（Webcam/WebAudio）のセキュリティポリシーに準拠するため、完全なセキュア通信を確保。

動作検証・デモ手順

  • Step 1: 画面下の「Start / 宇宙の波動と調和する」をクリック。
  • Step 2 (マイク拍動): マイクに向かってポンと手を叩き、白銀の衝撃粒子が一斉に弾けるのを確認。
  • Step 3 (感情変化): 表情を切り替える（Happy ➡️ Angryなど）。全身の関節から対応する色彩粒子が大爆発するのを確認。
  • Step 4 (手スワイプ): 手を素早く動かし、手の移動ベクトルの慣性を持った物理弾が射出されるのを確認。

08. Future Vision

KinetiChordが切り拓く将来の展望

1. VR/AR・XR空間展開

ウェブカメラの限界を超え、Apple Vision ProやMeta Questなどの3Dハンドトラッキングセンサーと統合。

空間の全方位（360度）に手から数万個の音響粒子を射出し、文字通り「音と光を全身に纏い、演奏する」メタバース・アートへ。

2. 外部DAW連携 (MIDI)

身体の動きの速度・感情値を、Web MIDI API経由でプロ仕様の音楽制作ソフト（Ableton Live, Logic Proなど）にリアルタイム送信。

ダンサーの「ダンスの激しさ」がシンセサイザーのフィルターを直接モジュレートする、究極の身体表現シンセ。

3. ライブステージ演出

リアルタイム生成される美しい星座アバターとスパークを、高輝度プロジェクターでダンスステージへ直接プロジェクションマッピング。

遅延わずか数ミリ秒で、演者の身体と劇的なリアルタイム視覚エフェクト・ライブ音響が完全共鳴する。
