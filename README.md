# 🌌 KinetiChord

「**KinetiChord（キネティコード）**」は、ウェブカメラから得られる**表情（感情）**や**身体のダイナミックな動き（Kinetic）**、そしてマイクで拾う**音声**を極限まで解析し、3D空間上のきらびやかな星座スケルトンアバターと**電子和音（Chord）**に変換して共鳴させる、プレミアムなデジタルアート作品です。

## ✨ 主要機能

### 1. 【動的追跡】キネティック・スケルトン（二律駆動アバター）
* **カメラ検出時**: 表情や手の位置をミリ秒単位でスムージング（Lerp）追跡。3Dの肉厚なテーパーシリンダーによって、美しい立体的な身体構造をスクリーンへリアルタイム投影します。
* **カメラ非検出時**: アバターが自動的に「自律遊泳ダンスモード」に切り替わり、呼吸するような2重調和振動（∞字ウェーブモーション）で生命力をたゆたわせます。

### 2. 【光の演出】3層のダイナミック・パーティクルシステム
* **① 手速度追跡＆物理軌跡バースト (Motion-Driven Tail Burst)**:
  * 左右の手の位置のフレーム間差分から移動速度をミリ秒単位で算出。
  * 素早くスワイプするような動作をした瞬間、手の軌跡の慣性を引き継いだ **3〜8個の美しく輝くバースト粒子** を進行方向に射出。アバターが文字通り「光の玉を投げ飛ばす」ような極めて直感的なインタラクションを提供します。
* **② 全身連動「モーションスパーク」 (Total Body Motion Sparks)**:
  * カメラ全体のピクセル差分から動きの激しさを算出し、動きが激しいとき（ダンス等）に全身の関節から**「まばゆく瞬き、超高速で四散する虹色（レインボー）の極小火花」**を毎フレーム大放出。
* **③ 星座ライン（接続線）の動的脈動**:
  * 人体構造トポロジーに基づき、隣接する骨格パーツのみを接続（クモの巣ノイズを100%排除）。全身の動きが活発になると、接続線の太さ・輝度がダイナミックに脈動し、最大接続距離が自動拡張されます。

### 3. 【感情共鳴】極限音響モジュレーション (Extreme Sound FX)
WebAudio APIのシンセサイザーと音声処理ノードをフル活用し、感情と100%シンクロした極限的な音響世界を構築します。

| 感情 | ビジュアル演出 | 音響モジュレーション効果（Extreme Sound FX） |
| :--- | :--- | :--- |
| **😊 喜び (Happy)** | 黄金（ゴールド）の粒子が優しく舞い上がる | **カスケード・アルペジエーター・スイープ**:<br>フィルターを9,500Hzまで全開。120ms周期でコード音がオクターブ上（+12/+24半音）へステップするキラキラとしたグリッサンド。 |
| **😢 哀しみ (Sad)** | 青い涙の粒子が静かに下へ滴り落ちる | **深海フェードアウト ＆ スペースエコー**:<br>音量を最大85%カット。ローパスを150Hzまで絞った極限マッフル状態にし、88%超高帰還のディレイで遠くへ溶けていく哀愁。 |
| **😡 怒り (Angry)** | 凄まじい初速で真紅の火花が吹き出す | **オーバーブースト・ディストーション**:<br>クリーン音を完全消滅。WaveShaperによる歪み量を極限（350）まで引き上げ、さらにマイク入力連動の爆音ゲインと高周波悲鳴ピッチジッターを付与。 |
| **😮 驚き (Surprised)** | シアンの衝撃波パルスが急激に飛び散る | **超高速15Hzステレオパン ＆ 金属調ショートエコー**:<br>左右の定位が1秒間に15回往復して脳を揺さぶり、0.05秒の極小金属的スラップバック・ディレイが響くサイケデリック音響。 |
| **😐 平穏 (Neutral)** | エメラルドグリーンの光の霧がたゆたう | **深みのある心地よいドローン音**:<br>サイン波と三角波による5声の美しい平均律の持続和音が安定して鳴り響きます。 |

> [!TIP]
> **手の位置連動ステレオ定位**: 
> 驚き感情が発生していないときは、左右の手の重心位置の差分に基づき、音像全体がその手の位置にダイナミックに定位します（左に行けば左、右に行けば右から音が聴こえます）。

## 🛠 技術スタック

* **フロントエンド**: HTML5, Vanilla CSS, Vanilla JavaScript (ES6+)
* **顔認識・表情解析**: `face-api.js` (TensorFlow.js ベース)
* **音響・シンセ合成**: `Web Audio API` (BiquadFilter, WaveShaper, Delay, Gain, StereoPanner, Oscillator)
* **アバター描画**: 2D Canvas (3D投影ベクトル数学モデリング・リアルタイム自転および俯瞰たゆたい旋回)
* **デプロイ・運用**: Google Cloud Run, Docker

## ⚙️ ローカルでの開発方法

### 1. リポジトリをクローン
```bash
git clone https://github.com/maa0917/KinetiChord.git
cd KinetiChord
```

### 2. ローカルサーバーを起動
HTML/JSで完結しているため、任意の静的サーバーで起動できます。

**VS Codeを使用する場合:**
「Live Server」プラグインなどで `index.html` を起動してください。

**Pythonを使用する場合:**
```bash
python3 -m http.server 8080
```
ブラウザで `http://localhost:8080` を開きます。

## 🐳 Docker ＆ GCP Cloud Run へのデプロイ

### ローカルでのDocker起動
```bash
# ビルド
docker build -t kinetichord .

# 起動
docker run -p 8080:80 kinetichord
```

### GCP Cloud Runへのデプロイ（参考）
```bash
gcloud run deploy kinetichord \
  --source . \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated
```
